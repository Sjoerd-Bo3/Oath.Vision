# Oath.Vision

A viewer and lineage tracker for Oath board games.

## Setup

`npm install`

## Developing

`npm start`

## Building

`npm run build`

## Assets

If the `postinstall` task fails, you'll need to run `npm run asset:pack` to build all of the asset files.
